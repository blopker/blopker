Pheme
Web-based user console for distributed systems.
Project site (Source)
Pheme allows non-technical users take control of distributed compute farms from their browsers. Pheme will originally be packaged with the distributed graph processing system jPregel, but is designed to work in any distributed system. Although Pheme runs in a JVM it has adapters that can hook into any communication mechanism. This is my master's thesis project at UCSB.
