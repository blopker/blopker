Pheme
Web-based user console for distributed systems
Project site (Source)
Pheme gets its name from the Greek goddess of "scandalous rumors". She has also been described as "she who initiates and furthers communication". This project aims to facilitate communication between users and distributed compute farms. To do this Pheme allows non-technical users to control these systems through their browsers. Pheme will start life packaged with the distributed graph processing system jPregel, but it is designed to work with any distributed system. Pheme has an adapter architecture that can hook seamlessly into any communication protocol.
Technology
Java and Scala, Play Framework 2.1, Websockets, Google's Guava Eventbus, Twitter Bootstrap, RMI (for jPregel communication)
Powwow Puzzler
Transmission power impact on congested network throughput
Paper [PDF] (Source)
Using 9 Android devices and 3 wireless APs we tested how antennae power affected wireless throughput. To accomplish this, we wrote an Android app that can send and receive UDP packets with a Java application on our server. We then varied transmission power on the devices and monitored total network throughput. Then we made recommendations on our findings.
Technology
Java, UDP Sockets, DD-WRT, 802.11, Android, Load Testing
QR Live
Augmented reality QR code interaction for Android
Project site (Source)
QR Live aims to consume QR codes as naturally as possible. This Android app first reads QR codes then recreates the code as a clean digital image. Then using augmented reality feature detection algorithms like GFTT (Good Features To Track) QR Live tracks the code in real time while overlaying the code's decoded information. Although this app is currently unfinished it is a good example of using AR technology to track arbitrary markers.
Technology
Android, ZXing, OpenCV, Augmented Reality
Semiconductor Simulator
Parallelized diode simulation
Project site (Source)
For this project I teamed up with two material engineers to simulate semi-conducting diodes. Using numerical analysis I was able to take the engineers' PDEs and produce parallelized simulation software. I used the high performance ACML linear algebra library with Armadillo bindings.
Technology
C++, Armadillo, ACML, Computer Modeling
KDD Music Recommender
Hadoop based music recommendation system
Paper [PDF] (Source)
A music reccomendation system for the Yahoo KDD cup. Used the K nearest neighbor algorithm on several gigabytes of music data to suggest new songs to users. Hadoop took care of parallelization and scaling.
Technology
Hadoop, Java, args4j, Data Mining, Machine Learning
